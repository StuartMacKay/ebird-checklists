This directory contains sample spreadsheets for data that can be downloaded
from eBird:

ebird_basic_dataset_sample.csv is a copy of the sample file for the eBird
Basic Dataset, available on the eBird web site, https://ebird.org/data/download
To visit this page you will need an eBird account. You will also have to request
access to download the actual data, which is updated on the 15th of each month

MyEBirdData.cvs shows the format of the records downloaded when you log into
your eBird account and visit the "Download My Data" page, https://ebird.org/downloadMyData
The file contains actual records from this project author's eBird account.
